# Return Policy

## Return Window

You have **14 days** from the date of delivery to return items for a full refund.

## Return Process

1. Log into your account
2. Navigate to Orders
3. Select the item you want to return
4. Print the return label
5. Ship the item back

## Return Conditions

- Items must be unused and in original packaging
- All tags and labels must be attached
- Electronics must be factory reset
